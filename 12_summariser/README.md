# PDF Summarizer

Streamlit-приложение для преобразования PDF в Markdown с фильтрацией `Meta Files`. При выключенном переключателе локальная модель Ollama исключает служебные страницы и страницы только с заголовком, не раскрывающие тему документа. Toggle `Import Images` управляет сохранением нетекстовых изображений; при выключении в Markdown попадает только текст.

## Основные файлы

### `scripts/app.py`

Центральный интерфейс загрузки PDF, настройки фильтрации и пути сохранения.

### `scripts/document_service.py`

Координирует сохранение загруженных PDF, извлечение содержимого и запись итогового Markdown.

### `scripts/pdf_service.py`

Извлекает текст и встроенные изображения, определяет текстовые изображения через OCR и сохраняет только нетекстовые.

### `scripts/content_filter_service.py`

Классифицирует страницы через Ollama и формирует Markdown только из тематических страниц, если мета-информация выключена.

### `scripts/settings.py`

Читает конфигурацию Ollama и определяет папку сохранения по умолчанию.

### `.env.example`

Пример локальной конфигурации Ollama. Скопируйте его в `.env`.
