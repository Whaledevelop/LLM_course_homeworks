# Анализ результатов

## Фильтрация и конспектирование

Переключатель `Meta Files` по умолчанию выключен. В этом режиме каждая страница сначала классифицируется моделью Ollama. В Markdown попадают только страницы, которые прямо раскрывают тему документа: с фактами, аргументами, определениями, методами, примерами, результатами или выводами.

Meta Files — титульные слайды, приветствия, представления людей, программа, организационные объявления, вопросы аудитории, благодарности, контакты, информация о спикере и реклама. Они исключаются.

Страница только с тематическим заголовком не является Meta File. Модель помечает её как `HEADER`, а приложение добавляет этот текст в Markdown как заголовок второго уровня без номера страницы. Страница только с названием документа и авторами остаётся Meta File. Если модель сомневается между тематическим заголовком и служебной страницей, она исключает страницу.

При включённом переключателе фильтрация не выполняется: в Markdown попадают все страницы PDF.

## Изображения

Toggle `Import Images` управляет импортом изображений. Если он выключен, приложение не запускает OCR, не извлекает изображения из PDF и не создаёт папку `<имя>_assets/images`. В Markdown попадает только текст.

Если `Import Images` включён, встроенное изображение проверяется OCR. Если OCR уверенно находит не менее пяти слов и 30 символов, изображение считается текстовым: распознанный текст добавляется к содержимому страницы, а файл изображения не сохраняется. Остальные изображения сохраняются в `<имя>_assets/images` и добавляются в Markdown как относительные ссылки.

Для стабильной фильтрации текстовых изображений в системе должен быть установлен Tesseract с языками `rus` и `eng`. Если Tesseract недоступен, изображение сохраняется как нетекстовое.

## Проверка результата

1. Убедитесь, что итоговый `.md` и папка `<имя>_assets/images` появились по выбранному пути.
2. Проверьте, что при выключенном `Meta Files` титульные, организационные и страницы только с заголовком отсутствуют в Markdown вместе с их изображениями.
3. Выключите `Import Images` и убедитесь, что в указанной папке не создаются папки ресурсов, а в Markdown нет ссылок на изображения.
4. При включённом `Import Images` проверьте, что изображения с обычным текстом представлены текстом в Markdown, а схемы, фотографии и диаграммы сохранены отдельными файлами и имеют ссылки в Markdown.
