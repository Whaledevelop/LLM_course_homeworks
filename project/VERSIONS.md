VERSION_0_1

Версия 0.1 - локальный учебный ассистент на базе Streamlit для работы с личной базой Markdown-заметок.

Основная функциональность:

- Пополнение базы знаний: создание новых Markdown-заметок с названием, темой, сложностью, комментарием и учебными метриками.
- Автоматическое предложение метаданных для новой заметки через LLM.
- Просмотр и редактирование заметок: поиск, фильтрация по теме, открытие содержимого, обновление и удаление записей.
- Ответы на вопросы по материалам: ассистент ищет релевантные фрагменты локальных заметок и формирует ответ только на основе найденного контекста.
- Проверка учебных заданий: сравнение ответа ученика с материалами базы, вердикт, обратная связь, ошибки и пропущенные пункты.
- Active Recall: генерация вопросов по выбранной заметке, проверка ответа и сохранение истории последних попыток.
- Observability: просмотр статистики и событий работы ассистента для анализа качества ответов и проверок.
- Индексация материалов через гибридный поиск: BM25, векторный индекс Chroma и reranking для подбора контекста.
- Поддержка локальных моделей через Ollama и структурированных LLM-ответов для генерации, проверки и извлечения метаданных.
- Набор тестов и evaluation-сценариев для проверки поиска, схем данных, синхронизации метаданных и LLM-контрактов.
